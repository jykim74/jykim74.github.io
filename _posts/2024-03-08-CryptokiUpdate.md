## Version 2.2.8 업데이트
- 라이선스창에서 무료 라이선스 발급 지원 (이메일 입력 필요함)

## Version 2.2.6 업데이트
- 각각의 기능창에 대한 아이콘 표시

## Version 2.2.4 업데이트
- Unload 오류 수정
- 툴바 아이콘 변경 및 순서 처리 개선
- Hash, Sign/Verify Enc/Dec UI 개선
- 객체 보기에서 각각의 값 편집 지원

## Version 2.2.2 업데이트
- CSR 생성에서 동일 RDN 지원
- Init/Update/Final 상태 표시 개선
- 파일 드래그 앤 드랍 지원

## Version 2.2.0 업데이트
- 다수 오류 수정 및 UI 개선

## Version 2.1.8 업데이트
- 인증서 정보 서명값 표시 비트스트링 BER 헤더 제거
- UI 개선

## Version 2.1.6 업데이트
- 인증서, CSR, 개인키 보기에서 KID 값 표시
- OpenSSL 라이브러리 3.5.3 변경
- UI 개선 및 다수의 오류 수정

## Version 2.1.4 업데이트
- CAMan ProfileMan 마우스 우클릭 지원
- Export 시 개인키 공개키 PEM 헤더 표시 오류 수정
- UI 개선

## Version 2.1.2 업데이트
- 인증서 CRL 에서 2038년 이후 시간처리 오류 수정

## Version 2.1.0 업데이트
- 객체 보기 기능 지원
- 유효 객체값 보기 설정 지원
- 초기 입력값 제거
- 폰트 설정 고정폭 제한 선택
- DH 파라미터 가져오기/내보내기 지원
- 다수 오류 수정 및 UI 개선

## Version 2.0.8 업데이트

## Version 2.0.6 업데이트

- UI 개선 및 오류 수정

## Version 2.0.4 업데이트
- HSM 관리 기능의 값 못읽는 오류 수정
